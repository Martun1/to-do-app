# 📝 To‑Do App (CLI)

A lightweight and fast command‑line to‑do list manager built in Python.  
Perfect for practicing file handling, CLI design, and simple data persistence.

---

- ➕ Add tasks  
- 📋 List tasks  
- ❌ Remove tasks  
- 💾 Persistent file storage

- File handling (read/write)
