# 📝 To‑Do App (CLI)

A clean, fast, and minimalistic command‑line To‑Do application built in Python.  
Designed to help you practice file handling, CLI design, and simple data persistence — all in one compact project.

---

## ✨ Features

- ➕ **Add tasks** with a single command  
- 📋 **List all tasks** in a clean, readable format  
- ❌ **Remove tasks** by index  
- 💾 **Automatic file saving** (persistent storage)  
- ⚡ **Instant execution** — no dependencies, no setup  

---

## 🛠 Technologies Used

- **Python 3**
- Basic file handling (read/write)
- Simple CLI interaction

---

## 📁 Project Structure

| File        | Description                     |
|-------------|---------------------------------|
| `todo.py`   | Main application logic          |
| `tasks.txt` | Stores all saved tasks          |
